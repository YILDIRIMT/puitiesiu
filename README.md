# puitiesiu
puitiesiu basic embedded system

### Dependencies

* `qemu`, `make`, `nasm`
  
### Build

```bash
git clone https://github.com/YILDIRIMT/puitiesiu
```
```bash
cd puitiesiu
```
```bash
cd src
```
```bash
cd boot
```
```bash
make
```
```bash
cd ..
```
```bash
cd main
```
```bash
make
```
```bash
make run
```
